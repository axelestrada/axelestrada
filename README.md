### Dinaco
Aplicación móvil IoT para monitorear el nivel de agua de tinacos en tiempo real mediante sensores conectados.

Permite consultar el porcentaje de agua disponible, recibir alertas de nivel bajo, visualizar historial de consumo y administrar múltiples dispositivos desde una aplicación móvil.

Tecnologías involucradas:
- Kotlin
- PostgreSQL 
- Spring Boot 
- IoT
- MQTT
- Desarrollo móvil y comunicación con dispositivos embebidos
